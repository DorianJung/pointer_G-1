# pointer G-1
This is the Pure Data patch for my pointer G-1 granular effects unit. The patch is inspired by [yann seznec's ys.granular patch](https://github.com/yannseznec/ys.granular), which I used as a foundation to figure out what I needed to make this work. The patch has undergone major changes to be usable in a live performance environment, as I wanted to build my own granular synthesis effect pedal with it.
